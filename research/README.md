# research

External references, comparisons, and curated notes for this project.

## Contents

- `LINKS.md` — quick links
- `templates/project-note.md` — template for repo/reference notes
- `github-projects/` — one file per external repo or reference

## What belongs here

- Projects that solved similar coordination problems
- Tools that inspired this repo’s manifest or branch models
- Ideas worth cherry-picking or explicitly avoiding

## What does not belong here

- Implementation plans or task specs
- Internal-only decisions (those belong in `docs/` or ADRs)
